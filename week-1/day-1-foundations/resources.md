# Day 1 — Resources

## Primary Reading
- [Python.org: An Informal Introduction](https://docs.python.org/3/tutorial/introduction.html)
  — The official Python tutorial, sections 3.1 and 3.1.1. Read up to "Strings".
- [Real Python: Variables in Python](https://realpython.com/python-variables/)
  — Well-written, beginner-friendly. Skim it.

## Videos
- [Corey Schafer: Python Tutorial for Beginners 1 — Strings](https://www.youtube.com/watch?v=k9TUPpGqYTo)
  — Good pacing, good examples. ~20 min.
- [Corey Schafer: Python Tutorial for Beginners 2 — Integers and Floats](https://www.youtube.com/watch?v=khKv-8q7YmY)
  — Follow-up to the one above. ~15 min.

## Quick Reference
- [Python Cheatsheet — Basics](https://www.pythoncheatsheet.org/cheatsheet/basics)

## If You Finish Early
- [Python.org: Built-in Types](https://docs.python.org/3/library/stdtypes.html)
  — More types than you need today. Just browse — don't try to memorize.
